Veterinaria – Frontend

Aplicación frontend desarrollada en React para la gestión de una veterinaria. Incluye componentes reutilizables, consumo de API mediante Axios y una interfaz construida con Material UI.

Tecnologías principales: React (Vite), Material UI, Axios y JavaScript ES6.

Instalación y ejecución: Se instalan las dependencias del proyecto y se ejecuta el servidor de desarrollo mediante los comandos estándar de Node.js.

Estructura general: El proyecto se organiza en carpetas como components, pages, services y theme, lo que permite una separación clara de responsabilidades.

Características:

Arquitectura basada en componentes.

Patrón List + Form para implementar CRUD de forma consistente.

Manejo centralizado de solicitudes HTTP.

Interfaz responsiva con soporte de modo oscuro.
